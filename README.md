## Overview
This repository contains Jupyter notebooks created while practicing Apache Spark and Hadoop
on a Google Cloud Dataproc cluster.

## Topics Covered
- Spark RDD operations
- DataFrame APIs
- Caching and persistence
- GroupBy and Reduce
- Spark SQL
- Handling dates and data types
- Working with HDFS and GCS

## Platform
- Google Cloud Dataproc
- Apache Spark
- Hadoop (HDFS)
- Jupyter Notebook
